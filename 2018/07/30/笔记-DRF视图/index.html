json_data = serializers.serialize(‘json’,goods)
json_data = json.loads(json_data)
return JsonResponse(json_data,safe=False)

查看源码可知,默认safe为True,如果传入为非dict会报错
为了允许非dict对象被序列化，请将safe参数设置为False。

现在View是继承DRF的APIView,我们可以调用更上层的View:
现在来继承mixins和generic一起用:
class GoodsListView(mixins.ListModelMixin,generics.GenericAPIView):

class GenericAPIView(views.APIView):
GenericAPIView继承APIView，封装了很多方法，比APIView功能更强大
class ListModelMixin(object):

GenericAPIView用的时候需要定义queryset和serializer_class
GenericAPIView里面默认为空
queryset = None
serializer_class = None

ListModelMixin里面list方法帮我们做好了分页和序列化的工作，只要调用就好了

from goods.serializers import GoodsSerializer
from .models import Goods
from rest_framework.response import Response
from rest_framework import mixins
from rest_framework import generics
class GoodsListView(mixins.ListModelMixin,generics.GenericAPIView):
‘商品列表页’
queryset = Goods.objects.all()
serializer_class = GoodsSerializer

def get(self,request,*args,**kwargs):
    return self.list(request,*args,**kwargs)

定义queryset和serializer_class,r后写get方法,会调用list,在list中才会去序列化或者分页.

drf的APIView、GenericView、viewsets和router的原理分析


APIview –> genericview 添加了比如分页\过滤\queryset\serializer_class等
genericview + mixins –>得到各种组合ListAPIView(完成get与list的绑定)
genericview + viewsetmixins –> 修改as_view,使用另一种绑定方式



parsers可以解析提交或发过来的各种类型的数据

response根据前端要求返回不同类型的数据 比如网页形式还是json形式等

笔记参考

View 自己获取QuerySet的数据然后自己序列化完成后使用django的response返回
APIView 使用DRF的serializers进行序列化,然后使用DRF的response返回
GenericView 封装了很多方法,只要传入model和serializer_class就可以完成序列化,但是不会主动调用返回
mixins 在mixins的list中会调用Response(serializer.data),所以需要重载get函数
ListAPIView 继承了mixins和genericview,所以传入model等就不要重载get函数
pagination_class和serializer_class一样设置

GenericViewSet 继承ViewSetMixin和GenericView,在ViewSetMixin中重载了as_view方法,所以修改的是url到get这个过程
get到list是mixins完成的,所以使用viewsets标准姿势:
class GoodsListViewSet(mixins.ListModelMixin,viewsets.GenericViewSet):

因为重载了as_view方法,所以url的编写也不一样了:
from rest_framework.routers import DefaultRouter
router = DefaultRouter()

#配置goods的url
router.register(r’goods’, GoodsListViewSet)
url(‘^’, include(router.urls)),

如果不用DefaultRouter:
good_list = GoodsListViewset.as_view({
‘get’:’list’,
‘post’:’create’
})

如果继承了mixins,就不要重载get了

  django自带了admin用于后台管理,但是在github上还有一个更好看、功能更强大的库来管理后台,叫xadmin,今天就来好好介绍一下xadmin的使用.

xadmin的安装

  通过文档我们可以发现，我们有两种方法可以进行安装。

  1. 使用pip install 安装

    pip install django-xadmin

  2. 下载源码安装

    需要安装requirements.txt中的依赖包

  我使用的是源码安装,到github搜索xadmin,然后下载源码,然后在项目的工程目录下新建一个extra_apps,把源码放在该目录下,注意extra_apps需要与apps一样在settings中设置。这里不在重复叙述了,然后手动安装requirements.txt中的依赖包.最后在INSTALLED_APPS中添加xadmin和crispy_forms.
  接下来配置url:

from extra_apps import xadmin
url(r’^xadmin/‘, xadmin.site.urls),

  然后执行migrations和migrate生成数据库表并创建superuser.就可以访问登录啦
